# make_gif.py
Proyecto en Python que genera un GIF animado usando la librería Pillow. Crea fotogramas desde cero (sin imágenes externas) y los combina en output.gif, con una animación estilo “pixel/space” (orb en movimiento y estrellas parpadeando). Incluye script ejecutable y configuración simple para personalizar tamaño, colores, frames y velocidad.
